    # 生命周期

    无状态Widget

    1.整个周期所定义的状态不会发生改变

    2.所创的函数只会运行一次

    有状态的Widget

    1.整个周期所定义的状态可以发生改变

    2.可以通过setState实现状态更新

    initState() // widget被创建时执行
Build() //构建渲染Widget,一旦使用setState 会再次构建
Dispose() // Widget被删除触发
